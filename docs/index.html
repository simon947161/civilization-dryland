Wagga Climate Adaptive Building / Building Climate OS

A transparent concept-stage dashboard for testing climate pressure, outdoor climate buffer sizing, intervention contribution, rule-based recommendations, and a copyable design-review prompt.

This tool is an early-stage climate-adaptive design exploration prototype. It does not replace NatHERS, BASIX, NCC, Section J, engineering, planning, hydraulic, mechanical, electrical, structural, or ESG compliance assessment.
