A small climate operating system for a public-facing prototype home.

The demo turns building form, shade, planting, and water-sensitive design into a simple interface that can be shared as a single static HTML file.

  • Shade-first envelope: external protection, deep thresholds, and reduced summer exposure.
  • Dryland landscape buffer: planting and ground treatment support outdoor comfort around the home.
  • Water-aware operation: stormwater, soil cooling, and shade trees are treated as connected systems.
  • Public communication: works without accounts, installs, external API calls, or external libraries.
