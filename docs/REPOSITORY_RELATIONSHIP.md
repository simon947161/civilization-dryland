# Repository Relationship

## Repository Role

`civilization-dryland` is a long-term research archive about dryland civilization, climate adaptation and the future of human settlement.

It provides historical, cultural and regional memory for ClimateOS thinking.

## Position in the Wider System

```text
Dryland Civilization Knowledge
        ↓
Climate Adaptation Wisdom
        ↓
ClimateOS / BuildingOS long-term worldview
```

## Relationship to Other Repositories

- `Simon` is the personal headquarters and project index.
- `climateos-knowledge-garden` stores raw notes and CRP that may later support this archive.
- `climateos-book` may publish selected mature ideas.
- `kunlun-epa-site` may use dryland knowledge for Kunlun ecological planning and public communication.
- `eco-agent-system` may translate some environmental patterns into computable decision-support logic.

## Boundary

This repository should focus on:

- dryland history
- settlement patterns
- climate adaptation wisdom
- desert-edge development
- ecological transition
- regional resilience
- civilization-scale sustainability thinking

Do not use this repository as the main home for:

- personal career materials
- raw weekly CRP notes
- software runtime design
- funding proposal workflows
- local Wagga-specific application files

## Operating Rule

Civilization Dryland is the historical memory layer.

It should preserve long-term lessons from dry environments and translate them into future climate adaptation thinking.
